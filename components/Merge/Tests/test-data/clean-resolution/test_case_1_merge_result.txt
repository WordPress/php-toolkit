
### 1. Merge Evaluation Test Suites
The paper "Evaluation of Version Control Merge Tools" by UoW provides a detailed methodology for testing merge tools.
It highlights scenarios where:
- Base Document: The original document before any changes.
- Modification A: A modification made in one branch.
- Modification B: A separate modification made in another branch.
- Result: The expected outcomeconflict resolution.
Examples are included.
