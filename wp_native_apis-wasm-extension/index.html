How to choose a manifest

latest/manifest.json always points at the newest published bundle. It is convenient for manual testing, but it moves whenever trunk publishes a new extension.

Commit manifests are immutable. Use the release-history links below for docs, bug reports, demos, and tests that need to keep loading the same bundle over time.
